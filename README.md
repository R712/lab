# map styling
**Preview** https://r712.github.io/lab/map-style-full

### 만들어진 이유와 과정
구글맵 api 기반으로 호출해서 단편적으로 쓰일 때 보기좋은 레벨로 단순 육지/바다/국경선만 표시한 styling json이 필요했다.
- 육지위에 올라갈 마커, 텍스트들이 잘 보이도록 바다보다 육지를 밝게 두고 전반적으로 연하고 단조롭게 정리
- 마커들에 마이크로 애니메이션을 추가해 마커를 통해 이어질 다음 액션을 궁금하게 하기
- 컬러들은 간단히 변경 테스트 가능하게 하기
- Figma에서 다른 UI들과의 정렬과 어울림 테스트를 쉽게 하기 위해 svg/png*2 다운로드 붙임
- 완성된 json을 제3자가 가져가기 좋게 하기

### 보완하면 좋을 점
- **Figma 플러그인으로 배포하면 좋겠다**
- 브라우저 로컬스토리지로 히스토리 관리 기능 추가
- 위와 같은 맥락으로 기본적으로 테마를 제공
- 지도 위치와 줌인 레벨도 조정 가능하게 추가
- 커스텀 텍스트 마커 ex: ⛽️, 🏥
- 장소 분류별 토글 제공 (시군구, 동포함, 산/공원, 랜드마크등)
